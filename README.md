#Abalone

##Logic

+Linked lists  
+Six way linked list  
+Property on each ball that points to the other six objects  
+Six nodes  
+If a node is 0 then thats the edge  
+Separate pointer that points to the firstmost marble  

```javascript

marble {
    column: 0,
    row: 2
}
```
